# YTDown Website

Landing page para o aplicativo YTDown - Download de vídeos YouTube.

## Sobre

Este é um site de apresentação do aplicativo YTDown, criado por estudantes para estudantes. O site basicamente explica a funcionalidade do app e fornece links para download.

## Tecnologias

- HTML5 semântico
- CSS3 moderno (variáveis, grid, flexbox)
- JavaScript vanilla
- Design responsivo
- GitHub Pages (hospedagem)

## Funcionalidades do Site

- Hero section com mockup do app
- Seção de missão (estudantes para estudantes)
- Features em grid
- Como usar (passo a passo)
- Download cards (Windows e código fonte)
- FAQ interativo
- Aviso legal sobre copyright
- Footer com links
